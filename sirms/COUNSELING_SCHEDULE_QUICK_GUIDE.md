# Counseling Schedule - Quick Reference Guide

## For Counselors

### How to Schedule a Counseling Session

#### Method 1: After Evaluation (Automatic)
1. Go to **Referral Evaluation**
2. Click **Evaluate** on a case
3. Select commission level and intervention (NOT VPF)
4. Submit evaluation
5. **System automatically redirects** to Counseling Schedule
6. Schedule modal opens with case pre-selected
7. Fill in date/time, location, notes
8. Click **Schedule & Notify**

#### Method 2: From Counseling Schedule Page (Manual)
1. Go to **Counseling Schedule** from sidebar
2. Find case in "Pending Counseling Cases" section
3. Click **Schedule Session** button
4. Fill in date/time, location, notes
5. Click **Schedule & Notify**

### Managing Sessions

#### View Sessions
- **Calendar View**: See all sessions in monthly calendar
  - Navigate months with arrow buttons
  - Click on session to view notes
  - Color-coded by status

- **List View**: See all sessions in table format
  - Shows student, case ID, date/time, location
  - View notes button
  - Quick action buttons

#### Update Session Status
- **Mark as Completed**: Click ✅ icon in list view
- **Mark as Missed**: Click ❌ icon in list view
- Notifications automatically sent to student

### Which Cases Go Here?

✅ **YES - Schedule in Counseling Schedule:**
- 1st Commission: Parent Conference, Counseling/Follow-up
- 2nd Commission: Parent Conference, Counseling/Follow-up
- 3rd Commission: Parent Conference, Counseling/Follow-up

❌ **NO - Goes to VPF Schedule (ESP Teacher):**
- Values Reflective Formation (VPF)

### Notifications

**When you schedule a session:**
- ✉️ Student receives: Date, time, location, case ID
- ✉️ Reporter (teacher) receives: Confirmation of scheduling

**When you mark as completed:**
- ✉️ Student receives: Confirmation of completion

**When you mark as missed:**
- ✉️ Student receives: Reminder to reschedule

### Tips

1. **Pre-fill location**: Common locations like "Guidance Office" save time
2. **Add detailed notes**: Helps track session objectives
3. **Use calendar view**: Better for avoiding scheduling conflicts
4. **Check pending cases**: Review regularly to avoid delays
5. **Mark status promptly**: Keep records up-to-date

### Keyboard Shortcuts
- **ESC**: Close any open modal
- **Calendar navigation**: Use arrow buttons or click month name

### Common Questions

**Q: What if I evaluated a case but don't see it in pending?**
A: Check if it was a VPF case - those go to ESP Teachers

**Q: Can I reschedule a session?**
A: Currently, mark as missed and create a new schedule

**Q: Where is the old Counseling Management page?**
A: Still available in sidebar for backward compatibility

**Q: Do I need to notify students manually?**
A: No, system automatically sends notifications

**Q: Can I see past sessions?**
A: Yes, in list view - filter by status or scroll through calendar
